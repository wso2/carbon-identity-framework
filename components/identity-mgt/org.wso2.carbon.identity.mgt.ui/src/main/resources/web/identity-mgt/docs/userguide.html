User Identity Management

This user guide describes the user identity management related options and configurations available in the WSO2 Identity Server.

You can use the left hand side "Configure" menu to complete the below mentioned configuration and options details.

Challenge Questions Management

The Challenge Questions are the questions that the system presents to users when they need to recover the password by themselves in a situation where they have forgotten the password to login.



Following section describes of adding challenge questions to the system by Administrator. These questions will be then shown in each users Account Recovery menu and user can select questions and provide answer to them.

As shown in the Figure 1 click on Configure tab > Users and Roles and then click on Add Questions which is shown under Manage Challenge Questions.



Figure 1 : Challenge Question Management

The Challenge Question Management page will be shown which is depicted in the Figure 2. From here you can specify a new challenge question set by clicking on the Add new challenge question set button at the top. By default there will be two challenge questions sets displayed. You can click them to see the challenge questions available for each set. Also you can delete existing challenge question sets by clicking the relevant delete icon as shown in the Figure 2.




Figure 2 : Challenge Question Main

Adding Challenge Questions to existing set

You can add more challenge questions to an existing set by clicking on a set shown in Figure 2 and the Add Challenge Questions screen will be displayed. As shown in Figure 3 the set id will be read only. The questions available for the selected set will be displayed in the Challenge questions table as shown in Figure 3.

A new question can be defined in the Enter new Challenge Question text box and click on Add button to add it to the challenge question set. After defining the questions you can click on Finish button to save the changes. If you don't want to save the changes click on Cancel and you will be directed to previous page.

You can delete the questions from the table using the correct delete icon. Click Yes in the delete confirmation dialog to remove it from the table. Click on Finish button to save the changes.


Figure 3 : Add Challenge Questions

Editing a Challenge Question

You can edit existing challenge question by clicking on the Edit icon displayed against the question. This will populate the text box with the question and remove it from the table as shown in the Figure 4. When you finish editing the question click on Update button and it will again add the question to the table. Click on Finish button to save the changes.




Figure 4 : Edit Challenge Question

Adding Challenge Question to new set

When you click on Add challenge question set from Figure 2 you will be directed to Add Question screen as shown Figure 5. Give a unique set id and start adding questions. After you complete click on Finish button to save the challenge question set.


Figure 5 : Add New Challenge Question set
